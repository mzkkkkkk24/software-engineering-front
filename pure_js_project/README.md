# MediaShare
**MediaShare** 是一个现代媒体社交分享平台，让用户轻松发现精彩、分享瞬间。支持图片、视频上传、内容发布、关注/粉丝系统、评论互动、内容评分等功能，打造属于你的媒体社交世界。

## 主要功能(用户页)

- **登录功能**：可根据账号身份判断跳转管理员或普通主页，避免页面冗余
- **用户系统**：注册、登录、编辑资料（头像、昵称、个性签名）
- **内容发布**：支持图片/视频上传、标题、正文、标签
- **社交互动**：关注/粉丝、评论、内容评分（星星系统）
- **个人主页**：展示用户发布内容、统计
- **动态背景与美观UI**：首页漂浮生活照片效果、深色模式切换、响应式设计
- **Toast 通知**：友好提示替换原生 alert
- **好友功能**: 好友功能支持添加删除
- **内容动态无限加载**: 主页内容卡片可以无限加载不需要点击跳转页面
- **聊天机器人接入**：接入聊天机器人模型，实现与ai实时对话
- **多模态搜索**: 可根据文字搜索相关图片内容，也可以通过图片搜索相关内容
- **本地存储代替服务器**: fileURL存本地路径
- **更多**：内容编辑/删除、菜单操作等

## 主要功能(管理员页)
- **对用户的管理**：支持对用户的删除、查询
- **对内容的管理**：实现对内容的模糊查询、删除
- **活跃度统计**：可以统计近七天用户活跃情况、采用条形统计图展示
